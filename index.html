Proxy Setup Required

SAGE's API (promoplace.com/ws/ws.dll/ConnectAPI) must be called server-side — direct browser calls are blocked by CORS. This is standard and expected.

The fix is a Cloudflare Worker proxy — a small script that sits between the dashboard and SAGE, forwarding requests securely. One-time setup, completely free on Cloudflare's free tier, takes about 10 minutes to deploy.
